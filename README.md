# Quantum Learning Journal
A personal journal documenting my journey through quantum computing, AWS Cloud, IoT/IIoT, and cybersecurity. This repository showcases hands‑on experiments, technical notes, and continuous learning toward quantum‑ready and quantum‑safe architecture.

## Purpose
This journal serves as a living record of my progress in:

- Quantum programming

- Quantum neural networks (QNNs)

- Quantum‑safe cryptography

- Cloud fundamentals

- IoT/IIoT security

It demonstrates applied learning, experimentation, and real‑world problem solving across emerging technologies.

## Technologies & Tools
- Python — PennyLane, Qiskit, NumPy, Matplotlib

- Quantum ML — QNNs, variational circuits, hybrid models

- AWS Cloud — EC2, S3, IAM, CloudShell

- IoT/IIoT — sensor simulation, MQTT, edge security

- Quantum‑Safe Security — lattice‑based concepts, hash‑based signatures

- Linux — CLI workflows, environment management

- Git & GitHub — version control, branching, documentation

Training Quantum Neural Networks (QNNs) using the Adam optimizer, analyzing convergence behavior, and exploring hybrid quantum‑classical learning workflows.

Recent work includes:

- Implementing 3‑qubit QNN models

- Plotting training loss curves

- Experimenting with circuit depth and optimizer parameters

- Documenting quantum‑safe concepts relevant to future IoT systems
